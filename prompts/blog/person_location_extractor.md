以下の文章に登場する「地名・国名・都市名・地域名」「人物名」「重要キーワード」を抽出し、JSONのみを出力せよ。

# 抽出の重要ルール（厳守）
1. **places（地名・地域名）**：
   - 特徴、歴史的背景を必ず「info」に含めてください。
   - 現在の国名だけでなく、**歴史的な呼称や地域名（例：ルアンダ＝ウルンディ、ザイール共和国など）**も漏らさず抽出してください。
   - ~語は国名にしない（フランス語、ドイツ語など）
   - 主要貿易相手国（トップ3）の情報があれば、それらも places に含めてください。
   - ※ 日本は除外してください。

2. **people（人物名）**：
   - 肩書き、経歴、この文章での役割を「info」に記述してください。
   - フルネームだけでなく、**役職（大統領、首相、少将、中将など）**が付随している場合も含めて特定してください。
   - 主要な歴史的人物は必ず含めてください（例：ジュベナル・ハビャリマナ少将、ジャスティン・ンセンギユムヴァ首相など）。

3. **keywords（キーワード）**：
   - 重要概念、制度、略語などの定義や文脈を「info」に記述してください。

4. **命名形式（厳守）**：
   - 記載する場所・国名は、必ず以下の形式で表記してください。
     形式：通称 

# 出力形式
- **JSON以外は絶対に出力しないこと（コードブロック ```json なども禁止）**。
- JSONのルートキーは「places」「people」「keywords」の3つとしてください。

# 出力テンプレート
{
  "places": [
    {"name": "国名や地名 (English Name)", "info": "説明"}
  ],
  "people": [
    {"name": "人物名", "info": "説明"}
  ],
  "keywords": [
    {"name": "キーワード", "info": "説明"}
  ]
}

# 抽出対象の文章
{{ $json.article }}
